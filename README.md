# testing rest-client gem 

# Description
In this project we test the [rest-client]("https://github.com/rest-client/rest-client") gem. We use the GET method to search a query in a browser.

# How to run
Type this in your command line:

```
ruby main.rb
```

## Used
* Ruby 2.6.3p62
